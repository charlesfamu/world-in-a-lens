Introduction#

Banshee template is a One-Page portfolio / agency template. It comes with some features that are explained below.

Portfolio#

Portfolio uses a jQuery plugin called MixItUp. This plugin allows to sort elements, filter it and show only items from filter that are selected. Portfolio also uses custom plugin for displaying portfolio item description, link and bigger image of miniature. For order to work, portfolio requires to be initialized. In your JS section / file add this: 

Banshee.initPortfolio();
Default syntax of portfolio
<ul id="portfolio-filters" class="portfolio-grid-controls">
	    <li class="filter" data-filter="all">
	        All
	    </li>
	    ...
	</ul>
	<div id="portfolio-items" class="portfolio-grid">
	    <div class="portfolio-item web" data-title="Natural" data-description="Lorem." data-largesrc="pat/to/img.jpg">
	        <div class="portfolio-item-inner">
	            <div class="portfolio-image">
	                <div class="portfolio-overlay">
	                    <div class="portfolio-details">
	                        <h3>Natural Food</h3>
	                        <h6>Webdesign, Video</h6>
	                    </div>
	                </div>
	                <img src="assets/img/demo/portfolio/1.jpg" alt="" />
	            </div>
	        </div>
	    </div>
	    ...
	</div>
Explain
Element Description
<ul class="portfolio-grid-controls"> Main element for portfolio items filters. ID can be changed, but if you change this, make sure you pass the right ID in portfolio initializer (in JS)
<li class="filter"> Filter control element. The desired filter category of each filter button should be entered as the data-filter attribute. One element can be holder of one filter
<div class="portfolio-grid"> Main container of all portfolio items. Portfolio items should be placed in this container.
<div class="portfolio-item"> It's a main container of each portfolio item. The filtering categories of each target element should be entered into its class attribute.
Explain of data attributes for expanding portfolio details:
data-title - Portfolio item title. Should be the same as title of element.
data-description - Description of portfolio item. Curenttly you can use only plain text.
data-largesrc - Path to bigger portfolio item image. Relative from the document.
data-link - URL to anywhere. It will be placed in portfolio detals button
<div clas="portfolio-details"> Title and categories of portfolio item
h3 - Portfolio item title
h6 - Portfolio item categories (filters)

Parallax#

Probably you know what parallax effect is, so I will not write about it. Focus on usage it in Banshee.
In order to parallax effect to be worked, you must add class parallax to element. This element should be a section, direct children of div#wrapper element.

Example
<section class="parallax" > ... </section>

To this element you must set the background via style attribute or via CSS. Position of background is calculated automatically.

Next is to add a one Javascript line:

Javascript code
$( '#some-element' ).parallax( horizontalPosition , speed , outerHeight );
Element Default Description
#some-element N/A This is main parallax container that effect will be applied to.
horizontalPosition 50% Horizontal position of background.
speed 0.1 Speed of background position change. Value should be within rnage 0.1 - 0.5 for the best effect.
outerHeight true By default, script will take a container height as outer height. Set it to false to take normal height.

Animations#

Banshee has a several animations that can be added to elements and it will show with animation when it appears on screen. It can be applied to most of html elements, such as divs, span, h3 and others. With this, when elements will appear on screen (will be visible in viewport) it will animate with chosen animation. Currently Banshee template has five animations and many new is planned to be added in future updates.

Animatons is applied to element by adding a data-animate attribute to desired element with name of animation as value of attribute. There is also available attribute data-delay if you want to run animation after delay of time or turn off delay. Default delay is 150 * number of element

Attribute Possible Values Description
data-animate fade-in
fade-in-top
fade-in-bottom
fade-in-left
fade-in-right 		In this attribute you must specify the name of animation that will be triggered.
data-delay Numeric (1000 is 1sec)
none 		This attribute is optional. Defaul value without this atrribute is 150 * number of element in order to animate. When is set to none element will animate right after appears in viewport. When value is numeric, then it will be added to the default delay.

Twitter Feed#

Twitter Feed (twitterfeed.js} is used to display latest tweet from desired Twitter profile. For work Twitter Feed requires a PHP support (use a external library twitteroauth written in PHP). Usage is very simple:

Example usage
$(document).ready(function(){
	    $('#twitterfeed').latestTweets({
	        twitterprofile: 'mytwitterprofile',
	        displaylimit: 4,
	        showavatar: true
	    });
	});

Here is complete list of available options:

Option Default Description
displaylimit 3 Number of tweets to get
showavatar false Set true to display avatars
showretweets false Set true to show retweets from profile
showscreename false Set true to show profile screen name
showtwittername false Set true to show twitter name (name followed by @)
showtweetlinks true If set to true, links will be displayed as anchors, if false, links will be displayed as plain text
showretweetindicator false Show a retweet indicator to display tweet as retweeted
